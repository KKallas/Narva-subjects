## 3D Printing and CAD: Lab 2

### Objective

Improvements to the initial syringe holder:

* 3D printed spring mechanism for syringe replacement: when the syringe runs empty, it must be replaceable easily and without tools.
* The syringe should be in as close to the same position as the previous syringe before replacement
* Make a modular tool holder so it's possible to print either a Dobot MG400 or UR5 mount for your syringe dispenser

**2nd mock-up objective** Replace the tool tip so we can begin testing with the UR5 robot. Create the ability for quick syringe replacement, since different adhesive materials are sold in pre-filled syringes — syringe replacement must be fast, convenient, and precise — because in production this is done more than 10 times per day.

### Requirements:

- [ ] The syringe must attach to the robot's tool tip.
- [ ] The syringe must be controllable from the robot's software.
- [ ] A pneumatic piston must be attachable to the syringe for automatic dispensing.
- [ ] The syringe must be replaceable easily and without additional tools.

#### Test Example 1: Fusion 360 Intersection Test

Before printing, verify that the virtual objects in the computer are properly constrained relative to each other and check that objects don't intersect each other.

#### Test Example 2: Basic Dimensions

After printing, you need at least one measurement in each axis — three measurements that are easy to check with calipers, to verify the finished object is to scale and not stretched.
